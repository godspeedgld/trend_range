# Interpretation

## ADF（Augmented Dickey-Fuller）

- H0：序列存在单位根（非平稳）；H1：平稳。
- p < 0.05 → 拒绝 H0，序列平稳。
- 用法：检测**收益率/差分**。价格序列一般非平稳，需先差分。
- 收益率通常已平稳；若收益率仍非平稳，警惕结构性断裂或数据问题。

## Ljung-Box

- H0：前 m 阶无自相关；H1：存在自相关。
- 任一阶 p < 0.05 → 判定存在自相关。
- 作用在原始序列：判断有没有可建模的线性动态。
- 作用在**残差/标准化残差**：判断模型是否把线性动态吸收干净（p > 0.05 才算通过）。

## ARCH-LM

- H0：无 ARCH 效应（条件方差恒定）；H1：存在波动聚集。
- 任一阶 p < 0.05 → 存在 ARCH 效应，需要 GARCH。
- 直观证据：收益率平方序列出现成簇的高低起伏（波动聚集）。

## 方差比检验 VR（Lo-MacKinlay）

- H0：序列服从随机游走（VR(q) = 1）。
- VR(q) = Var(q 期收益) / (q · Var(1 期收益))，H0 下 = 1。
- VR > 1 → 正自相关（动量）；VR < 1 → 负自相关（均值回复）。
- 任一阶 p < 0.05 → 拒绝随机游走。
- 与 Ljung-Box 重叠但视角不同：VR 是多持有期联合判断，是"市场有效/随机游走"的标准检验。
- 当 LB 与 ARCH 都不显著时，VR 用于正式判定"是否随机游走"，决定走漂移基线还是仍有非线性结构。

## RandomWalk（随机游走 / ARIMA(0,1,0) with drift）

- 模型：价格 P_t = P_{t-1} + μ + ε_t；在收益率空间即 r_t = μ + ε_t（常数漂移 + 白噪声）。
- 等价于 ARIMA(0,1,0) with drift，是最朴素的时序模型。
- 适用：无自相关、无 ARCH 效应，且方差比检验不拒绝随机游走。
- 最优预测：明天价格 = 今天价格 + μ；μ = 训练集收益均值。
- 拟合充分性：残差（去均值后）Ljung-Box 通过 → 随机游走假设成立。
- 注意：随机游走是"一等公民模型"，不是"无模型"。趋势行情里漂移外推短期往往看着准，
  但那是趋势延续，并非真正"可预测结构"——一旦行情转折即失效。

## ARMA

- 对**条件均值**建模：用历史值线性预测当期。
- 阶数 (p, q)：p 自回归项、q 滑动平均项，由 AIC/BIC 在网格上选最小。
- 适用：有自相关、但方差稳定（无 ARCH 效应）。
- 拟合充分性：残差 Ljung-Box 全部 p > 0.05。

## AR + GARCH

- 均值方程用 AR(p)（条件均值有线性动态，但 ARMA 的 MA 项不显著），
  方差方程用 GARCH(p,q) 描述波动聚集。
- 适用：无显著自相关（或仅 AR 可吸收）但存在 ARCH 效应。
- 拟合充分性：标准化残差 LB（均值）通过、标准化残差平方 LB（方差）通过。

## ARMA + GARCH

- 均值方程 ARMA(p,q) + 方差方程 GARCH(p,q)，同时刻画线性动态与波动聚集。
- 适用：既有自相关、又有 ARCH 效应。
- 本 skill 采用**两步联合估计**：先 ARMA 拟合取残差，再在残差上拟合 GARCH；
  标准化残差 = ARMA 残差 / 条件标准差。
- 拟合充分性：标准化残差 LB 与其平方 LB 同时通过。

## 通过 / 未通过

- RandomWalk：残差（去均值后）无可检测自相关 → 通过（随机游走成立）。
- ARMA：残差无可检测自相关 → 通过。
- GARCH 类：均值方程与方差方程 LB 同时无自相关 → 通过。
- 未通过通常意味着阶数偏低、分布假设不合适，或存在非线性/结构断点。

## 策略与因子含义（研究方向，非下单）

- ARMA 显著 → 短期可预测性，可研究短线动量/反转、均值回复。
- GARCH 显著 → 波动聚集，可研究波动率择时、期权定价、尾部风险控制、
  用条件标准差对收益标准化（如 TSMOM 的 σ_t）。
- ARMA+GARCH 通过 → 序列的线性动态被较好刻画，残差近似白噪声。
