# Interpretation

## ADF（Augmented Dickey-Fuller）— 前提

- H0：序列存在单位根（非平稳）；H1：平稳。
- p < 0.05 → 平稳。本 skill **以此为前提**：非平稳直接抛 `NonStationaryError`。
- 作用对象：**收益率/差分**。价格序列一般非平稳，需先差分。

## Ljung-Box — 均值方程（短期自相关）

- H0：前 m 阶无自相关；任一阶 p < 0.05 → 存在自相关。
- 作用在原始序列：判断有没有可建模的线性动态（→ ARMA）。
- 作用在**残差/标准化残差**：判断模型是否把线性动态吸收干净（p > 0.05 才算通过）。

## GPH 长记忆 — 均值方程（分数积分 d）

- 估计分数差分参数 d：对周期图低频点做 `log I(ω) ~ log[4 sin²(πf)]` 回归，d = -斜率。
- d=0 短记忆；d∈(0,0.5) 长记忆（持续性，ACF 双曲衰减）；d∈(-0.5,0) 反持久。
- `has_long_memory`：`|d|>0.1` 且 p<0.05 → 建议均值方程用 **ARFIMA**（两步法）。
- 均值方程判定以 **Ljung-Box 为总闸**：无自相关即判 Constant；仅当 LB 显著时才用 GPH
  区分长记忆(ARFIMA)与短记忆(ARMA)。GPH 在 ARFIMA 与 ARMA 之间二选一，不越过 LB 的门槛。

## ARCH-LM — 方差方程（波动聚集）

- H0：无 ARCH 效应（条件方差恒定）；任一阶 p < 0.05 → 存在 ARCH，需要 GARCH/GJR。
- 直观证据：收益率平方序列成簇的高低起伏（波动聚集）。

## Engle-Ng 符号偏差 — 方差方程（杠杆 / 非对称）

- 回归 `ε² ~ const + S⁻ + S⁻·ε₋₁ + (1−S⁻)·ε₋₁`（S⁻=1 若 ε<0）。
- `negative_p`（负向尺寸偏差，经典杠杆）、`positive_p`（正向）、`joint_p`（联合 F 检验）。
- 任一 p < 0.05 → `has_asymmetry=True` → 建议方差方程用 **GJR-GARCH**（而非对称 GARCH）。
- 杠杆效应：负冲击比等量正冲击更抬升未来波动（典型于股票/部分商品）。
- 检测阶段尚未建模，用 `r - mean(r)` 作为 ε 近似。

## ACF / PACF — 辅助证据

- ACF 拖尾、PACF 截尾 → 偏 AR；PACF 拖尾、ACF 截尾 → 偏 MA；均拖尾 → ARMA。
- 报告中 `*` 标记超出 95% 置信区间的显著阶，辅助人工复核均值阶数选择。

## 均值方程模型

### Constant
- μ = 样本均值，残差 = r - μ。无自相关、无长记忆时使用。

### ARMA
- 对**条件均值**建模：历史值线性预测当期。阶 (p,q) 由 AIC/BIC 网格选最小。
- 适用：有自相关、方差稳定（无 ARCH）。拟合充分性：残差 LB 全部 p > 0.05。

### ARFIMA（两步法）
- 长记忆模型。两步：GPH 估 d → 分数差分 (1-L)^d → 差分序列上 fit ARMA。
- 阶 (p,d,q)。创新 = 差分序列上的 ARMA 残差；其 LB 判定拟合充分性。
- arch/statsmodels 不支持分数 d，故自实现（权重截断 k，丢弃边界点）。

## 方差方程模型

### GARCH（对称）
- σ²_t = ω + α·ε²_{t-1} + β·σ²_{t-1}；描述对称波动聚集。
- 适用：有 ARCH、无杠杆。阶 (P,Q) 由 AIC 选。

### GJR-GARCH（非对称 / 杠杆）
- σ²_t = ω + α·ε² + γ·S⁻ε² + β·σ²；`o=1` 的 gamma 项刻画杠杆。
- 适用：Engle-Ng 检出符号偏差。**注**：arch 8.0 的 EGARCH 无符号项，不能刻画杠杆，
  故本 skill 非对称一律用 GJR-GARCH。

## 三流程

- **flow_a**（均值 + 不变方差）：ARMA/ARFIMA → 残差 LB。
- **flow_b**（常数均值 + 方差）：GARCH/GJR 在去均值残差上拟合 → 标准化残差双 LB。
- **flow_c**（均值 + 方差）：迭代两步——① 定均值阶取残差 → ② 残差上定方差阶 →
  ③ 固定方差阶、按 `mean_AIC+var_AIC` 重选均值阶（至稳定）→ ④ 最佳阶最终拟合 → 标准化残差双 LB。
- **white_noise**（常数均值 + 不变方差）：不建模。

## 通过 / 未通过

- flow_a：创新残差无可检测自相关 → 通过。
- flow_b / flow_c：标准化残差 LB 与其平方 LB 同时无自相关 → 通过。
- 未通过通常意味着阶数偏低、分布假设不合适，或存在非线性/结构断点。

## 策略与因子含义（研究方向，非下单）

- ARMA/ARFIMA 显著 → 短期可预测性，可研究短线动量/反转、均值回复；ARFIMA 提示长期记忆/慢衰减。
- GARCH/GJR 显著 → 波动聚集 / 杠杆，可研究波动率择时、期权定价、尾部风险控制、
  用条件标准差对收益标准化（如 TSMOM 的 σ_t）；GJR 的 gamma 可作非对称风险因子。
- flow_c 通过 → 线性动态与波动结构被较好刻画，残差近似白噪声。
